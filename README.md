## 策略梯度
策略梯度是一種直接以神經網路產出**動作(action)**的建模邏輯，就像是我們人類在操作電腦一樣。在這樣的情境下，我們會將所謂的策略分成**行為策略(behavior policy)**和**目標策略(target policy)**來看待：行為策略表示被用於與環境互動以產生經驗的
